# 📖 Overview

TaskFlow Pro is a modern project management application that enables individuals and teams to manage projects, organize tasks, collaborate with team members, track progress, and analyze project performance in one centralized platform.

Designed with a clean UI and scalable architecture, TaskFlow Pro demonstrates industry-standard software engineering practices, responsive design, secure authentication, real-time collaboration, and analytics dashboards.

# ✨ Key Features

- 🔐 Secure OTP Authentication
- 📁 Project Management
- 📋 Kanban Task Board
- 📅 Calendar Scheduling
- 👥 Team Collaboration
- 📊 Analytics Dashboard
- 📈 Live Project Statistics
- 🔔 Notifications
- 🔍 Smart Search
- 📄 PDF & Excel Reports
